### 问题1：为什么需要残差？

**问题**：为什么深层网络难训练？

**原因分析**：
```
1. 深层网络梯度消失/爆炸
2. 梯度无法有效传播
3. 训练困难
4. 需要解决梯度问题
```

**对比**：

| 网络类型 | 梯度传播 | 训练难度 | 问题 |
|----------|----------|----------|------|
| **普通网络** | 困难 | 很难 | 梯度消失 |
| **残差网络** | 容易 | 容易 | **需要残差** |

**结论**：需要残差连接来解决梯度问题

### 问题2：如何实现？

**问题**：如何实现残差连接？

**解决**：
```
1. 输入 = 输出 + 输入
2. 恒等映射
3. 梯度可以直接传播
```

### 问题3：如何优化？

**问题**：如何优化残差连接？

**解决**：
```
1. Pre-LN: 残差在归一化前
2. Post-LN: 残差在归一化后
3. Pre-LN 更稳定
```

## 第三层推导：实现细节

### 3.1 残差实现

**问题**：如何实现残差？

**解决**：
```
1. 输入 x
2. 计算 F(x)
3. 输出 = F(x) + x
4. 梯度可以直接通过 x 传播
```

### 3.2 梯度传播

**问题**：梯度如何传播？

**解决**：
```
1. 残差连接提供直连通道
2. 梯度可以直接传播
3. 避免梯度消失
```
