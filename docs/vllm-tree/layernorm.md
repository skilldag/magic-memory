### 问题1：为什么需要归一化？

**问题**：为什么层间分布不稳定？

**原因分析**：
```
1. 深层网络层间分布变化大
2. 梯度不稳定
3. 训练困难
4. 需要稳定分布
```

**对比**：

| 归一化 | 优点 | 缺点 | 问题 |
|--------|------|------|------|
| **BatchNorm** | 稳定 | 依赖 batch | batch 小时不稳定 |
| **LayerNorm** | 稳定 | 不依赖 batch | **需要 LayerNorm** |

**结论**：需要 LayerNorm 来稳定层间分布

### 问题2：如何实现？

**问题**：如何实现归一化？

**解决**：
```
1. 计算均值和方差
2. 归一化
3. 缩放和偏移
4. 可学习参数
```

### 问题3：如何优化？

**问题**：如何优化 LayerNorm？

**解决**：
```
1. RMSNorm: 简化计算
2. Pre-LN: 放在残差前
3. 性能更好
```

## 第二层推导：LayerNorm 结构

### 2.1 LayerNorm 详细推导

```
[LayerNorm]
    │
    ├── 问题：如何计算？
    │       解决：计算均值和方差
    │
    ├── 问题：如何归一化？
    │       解决：(x - μ) / σ
    │
    └── 问题：如何可学习？
            解决：γ 和 β 参数
```

**为什么叫 LayerNorm？**
- 层（layer）归一化
- 按层归一化，不依赖 batch

### 2.2 RMSNorm 详细推导

```
[RMSNorm]
    │
    ├── 问题：如何简化？
    │       解决：只计算 RMS
    │
    ├── 问题：为什么更好？
    │       解决：计算更快
    │
    └── 问题：如何实现？
            解决：x / RMS(x)
```

**为什么叫 RMSNorm？**
- Root Mean Square
- 基于均方根的归一化

## 第三层推导：实现细节

### 3.1 归一化计算

**问题**：如何计算归一化？

**解决**：
```
1. μ = mean(x)
2. σ² = var(x)
3. σ = sqrt(σ² + ε)
4. y = (x - μ) / σ
5. y = γ * y + β
```

### 3.2 可学习参数

**问题**：为什么需要可学习参数？

**解决**：
```
1. γ: 缩放参数
2. β: 偏移参数
3. 训练时学习
4. 恢复信息
```
