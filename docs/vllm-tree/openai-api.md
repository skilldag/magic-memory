### 问题1：为什么需要兼容？

**问题**：为什么需要兼容 OpenAI？

**原因分析**：
```
1. OpenAI 是事实标准
2. 很多工具基于 OpenAI API
3. 用户希望无缝切换
4. 需要兼容性
```

**解决**：OpenAI API 兼容

### 问题2：如何映射？

**问题**：如何映射 OpenAI 接口？

**原因分析**：
```
1. OpenAI 有特定接口
2. vLLM 需要映射到自己的实现
3. 需要参数转换
4. 需要响应格式统一
```

**解决**：接口映射

### 问题3：如何扩展？

**问题**：如何扩展 OpenAI API？

**原因分析**：
```
1. OpenAI API 有局限
2. vLLM 有额外功能
3. 需要扩展接口
4. 需要保持兼容
```

**解决**：扩展字段
