```
  [Model 体系]
```

### 问题1：为什么需要 ModelRegistry？

**问题**：系统如何知道有哪些模型可用？

**原因分析**：
```
1. vLLM 支持多种模型（LLaMA, GPT, Mistral等）
2. 每种模型有不同的配置
3. 用户需要指定模型名称
4. 系统需要知道这个名称对应什么
```

**解决**：ModelRegistry（模型注册表）

**为什么叫 Registry？**
- 注册表，记录所有可用模型
- 类似服务注册，按名称查找

### 问题2：为什么需要 ModelLoader？

**问题**：模型文件如何加载到内存？

**原因分析**：
```
1. 模型权重存储在磁盘文件中
2. 需要读取并加载到GPU内存
3. 不同格式（safetensors, bin, pt）需要不同处理
4. 需要验证权重完整性
```

**解决**：ModelLoader（模型加载器）

**为什么叫 Loader？**
- 负责加载（load）操作
- 从磁盘到内存

### 问题3：为什么需要 Model？

**问题**：加载后的权重如何变成可执行的计算？

**原因分析**：
```
1. 权重只是数据
2. 需要包装成可执行的对象
3. 需要提供前向传播接口
4. 需要管理模型状态
```

**解决**：Model（模型对象）

**为什么叫 Model？**
- 模型本身，包含权重和计算逻辑

### 问题4：为什么需要 ModelRunner？

**问题**：模型如何高效执行？

**原因分析**：
```
1. 模型执行需要管理GPU资源
2. 需要处理输入/输出
3. 需要管理KV Cache
4. 需要处理批处理
```

**解决**：ModelRunner（模型运行时）

**为什么叫 Runner？**
- 负责运行（run）操作
- 管理执行过程

## 第二层推导：详细问题链

### 2.1 ModelRegistry 详细推导

```
[ModelRegistry]
    │
    ├── 问题：如何注册模型？
    │       解决：装饰器/注册函数
    │
    ├── 问题：如何查找模型？
    │       解决：按名称查找
    │
    └── 问题：如何获取模型配置？
            解决：返回配置对象
```

**问题：为什么需要注册？**
- 动态发现模型
- 不需要硬编码模型列表
- 支持插件式添加新模型

### 2.2 ModelLoader 详细推导

```
[ModelLoader]
    │
    ├── 问题：如何处理不同格式？
    │       ├── safetensors → 专用加载器
    │       ├── bin → HuggingFace加载器
    │       └── pt → PyTorch加载器
    │
    ├── 问题：如何验证权重？
    │       解决：SHA256校验
    │
    └── 问题：如何映射到GPU？
            解决：自动设备映射
```

**问题：为什么需要多种加载器？**
- 不同框架/格式需要不同处理
- 统一接口，内部多态

### 2.3 Model 详细推导

```
[Model]
    │
    ├── 问题：如何组织权重？
    │       解决：按层组织
    │
    ├── 问题：如何提供前向传播？
    │       解决：forward()方法
    │
    └── 问题：如何管理状态？
            解决：state_dict
```

**问题：为什么需要 Model 对象？**
- 封装权重和计算逻辑
- 提供统一接口
- 便于管理

### 2.4 ModelRunner 详细推导

```
[ModelRunner]
    │
    ├── 问题：如何管理输入？
    │       解决：输入预处理
    │
    ├── 问题：如何管理KV Cache？
    │       解决：KV Cache管理
    │
    ├── 问题：如何处理批处理？
    │       解决：动态批处理
    │
    └── 问题：如何管理GPU？
            解决：显存分配
```

**问题：为什么需要 Runner？**
- Model 只负责计算
- Runner 负责执行环境
- 关注点分离

## 第三层推导：Weights Loading

### 3.1 Weights Loading 详细推导

```
[Weights Loading]
    │
    ├── 问题：权重从哪来？
    │       解决：HuggingFace Hub / 本地文件
    │
    ├── 问题：如何下载？
    │       解决：缓存机制
    │
    ├── 问题：如何分片加载？
    │       解决：sharded weights
    │
    └── 问题：如何量化加载？
            解决：量化权重加载
```

**问题：为什么需要分片加载？**
- 大模型权重太大
- 单文件加载太慢
- 多文件并行加载
