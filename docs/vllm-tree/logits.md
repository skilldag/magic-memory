### 问题1：模型输出是什么？

**问题**：为什么叫 Logits 而不是概率？

**原因分析**：
```
1. 模型输出的是原始分数
2. 还没有经过 Softmax
3. Logits 是未归一化的对数几率
4. 需要转换成概率
```

**解决**：Logits（对数几率）

### 问题2：如何使用？

**问题**：Logits 如何变成 token？

**原因分析**：
```
1. Logits 需要转换成概率
2. 然后根据概率采样
3. 需要经过 Softmax
4. 需要采样策略
```

**解决**：Softmax + Sampling
