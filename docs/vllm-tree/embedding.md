# Embedding 深层推导

> 从"Token ID如何变向量"往下推，推到不能再分

---

## 表层问题

**输入**：Token ID 1234
**输出**：向量 [0.1, -0.2, 0.5, ...]
**问题**：如何把ID变成向量？

---

## 第二层推导：位置编码

```
[绝对位置]
    │
    ├── Sinusoidal

```

    │
    └── 问题：长序列泛化？

## 第三层推导：实现细节

### 3.1 词嵌入矩阵

```
shape: [vocab_size, hidden_dim]

存储:
    │
    ├── 问题：太大怎么办？
    │       解决：投影
    │
    └── 问题：并行查表？
            解决：gather操作
```

### 3.2 位置编码计算

```
正弦编码公式:
pos_emb[pos, 2i] = sin(pos / 10000^(2i/d))
pos_emb[pos, 2i+1] = cos(pos / 10000^(2i/d))
```
