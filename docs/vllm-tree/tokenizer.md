## 第二层推导：分词模式

### 2.1 BPE (Byte-Pair Encoding)

```
[BPE]
    │
    ├── 问题：如何合并高频字节对？
    │       解决：统计频次 → 合并
    │
    └── 问题：合并顺序？
            解决：从高频到低频
```

### 2.2 WordPiece

```
[WordPiece]
    │
    ├── 问题：如何选择子词？
    │       解决：最大化似然 → 选择
    │
    └── 问题：未登录词？
            解决：逐字符回退
```

### 2.3 SentencePiece

```
[SentencePiece]
    │
    ├── 问题：如何处理空格？
    │       解决：统一替换 ╎ _
    │
    └── 问题：如何端到端？
            解决：直接学tokenization
```

## 第三层推导：实现细节

### 3.1 BPE 实现步骤

```
1. 按字符分割 → 字节序列
2. 统计所有相邻对频次
3. 合并频次最高的对
4. 更新词表
5. 重复直到达到词表大小
```

### 3.2 Byte Fallback

```
未登录词处理:
    │
    ├── 问题：词表没有这个词？
    │       解决：拆成字节
    │
    └── 问题：还是无法表示？
            解决：<unk> 标记
```
