## 已收录知识

### vLLM 架构 (vllm-architecture.md)

基于 vLLM 技术架构，从易到难分为三个级别：

| 级别 | 数字范围 | 难度 | 内容 |
|------|----------|------|------|
| Level 1 | 0-9 | ⭐ 入门 | 基础设施和基础概念 |
| Level 2 | 10-29 | ⭐⭐ 中级 | 模型执行、缓存、注意力机制 |
| Level 3 | 30-50 | ⭐⭐⭐ 高级 | 优化、调度、服务化特性 |

## 记忆原理

**锚点连接法**：

```
数字 → 锚点图像 → 概念 → 记忆故事
```

例如：
- **16** = 石榴 → PagedAttention → 石榴多籽 = 多分页管理

## 如何使用

1. **查表**：遇到需要记忆的概念，查找对应数字
2. **回忆锚点**：看到数字想到锚点图像
3. **联想概念**：通过锚点想到对应概念
4. **串联故事**：多个概念用数据流路径串联
